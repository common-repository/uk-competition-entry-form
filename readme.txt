Adds a widget which gives your UK site visitors the chance to win fantastic prizes. Prizes updated regularly.The widget uses the email2win.co.uk site to provide content and generates the content from that site.

== Description ==

After installing and activiating the plugin you will see a new widget in your widgets called Competitions for prizes. Simply add a title and your site now offers fantastic prizes.The widget uses the email2win.co.uk site to provide content and generates the content from that site.
== Installation ==

1. Install the plugin
2. Activate the plugin
3. Go to Appearance > Widgets
4. Add the Competitions Widget
5. Enter a title
6. Click Save
7. Your site now has amazing prizes to be won.

== Frequently Asked Questions ==

= Do I need to offer prizes to be won ? =

No, this is all managed independantly through third parties. Your users are entering third party competitions.

= Do the users leave my site ? =

No. The user is never directed away from your website.
